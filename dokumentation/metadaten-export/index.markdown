```Metadata Export```

<br/>

**Deutsch:**

Um die Interoperabilität zwischen Systemen zu ermöglichen, müssen Daten in standardisierte Formate überführt werden. Diese Standards erlauben es, dass die eingelieferten Daten auch über das eigene System hinaus auffindbar, zugänglich und verwendbar sind.

<br/>

---

<br/>

**English:**

To enable interoperability between systems, data must be converted into standardized formats. These standards allow for the ingested data to be findable, accessible, and reusable beyond the boundaries of the source system.

<br/>
<br/>

## Rosetta-METS

```Rosetta METS```

<br/>

**Deutsch:**

Zur Übertragung unserer Metadaten an die [Landesinitiative Langzeitverfügbarkeit (LZV.nrw)](https://www.lzv.nrw/) im Hochschulbibliothekszentrum NRW verwenden wir das Format Rosetta-METS. Eine Spezifikation, mit der Umsetzung von zentralen Feldern des [arkumu.nrw-Datenmodells](/documentation/data-model/), finden Sie [auf der folgeden Seite](/documentation/metadata-export/Rosetta-METS). Bitte beachten Sie auch die öffentlichen Dokumentationen des Standards bei [ExLibris](https://knowledge.exlibrisgroup.com/Rosetta) und im [Wiki für Kunden und Partner des hbz](https://service-wiki.hbz-nrw.de/spaces/LLZV/pages/1205174424/Datenmanagement+in+der+LZV).

<br/>

---

<br/>

**English:**

We use the Rosetta-METS format to transfer our metadata to the [Landesinitiative Langzeitverfügbarkeit (State Initiative for Long-Term Preservation) (LZV.nrw)](https://www.lzv.nrw/) at the "Hochschulbibliothekszentrum NRw" (University Library Center NRW). A specification with the implementation of central fields of the [arkumu.nrw data model](/documentation/data-model/) can be found [on the following page](/documentation/metadata-export/Rosetta-METS). Please also refer to the public documentations of the standard at [ExLibris](https://knowledge.exlibrisgroup.com/Rosetta) and in the [Wiki for Customers and Partners of the hbz](https://service-wiki.hbz-nrw.de/spaces/LLZV/pages/1205174424/Datenmanagement+in+der+LZV).
